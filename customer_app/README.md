# Ridoo Customer App

Flutter application for Ridoo ride booking platform.

## Features

- Splash & onboarding
- Authentication (login, register, OTP)
- Home with map-based pickup/drop selection
- Ride booking & live tracking
- Wallet & payments
- Ride history, ratings, SOS, support

## Setup

```bash
flutter pub get
flutter run
```

Configure API URL in `lib/core/config/app_config.dart`.
